# Yoga Studio Landing Page

A modern, responsive landing page for a yoga studio built with React, Tailwind CSS, and React Router.

## Features

- **Responsive Design**: Built with Tailwind CSS for a mobile-first, responsive experience
- **React Router**: Client-side routing for smooth navigation between pages
- **Multiple Pages**: Home, About, Classes, and Contact pages
- **Modern Stack**: React 19, Vite, and Tailwind CSS v4

## Pages

- **Home**: Welcome page with studio introduction and key features
- **About**: Information about the studio's history and mission
- **Classes**: Showcase of available yoga classes with descriptions and details
- **Contact**: Contact information and message form

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Lint

```bash
npm run lint
```

## Tech Stack

- **React** 19.2.0
- **React Router DOM** 7.10.1
- **Tailwind CSS** 4.1.17
- **Vite** 7.2.4
